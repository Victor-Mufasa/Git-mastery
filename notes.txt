When you compress a folder to a zipped folder its contents size reduces to 15-25% its original size.
A 7zip folder with contents consumes less space.
A gzip file also reduces the file size but not as much as a 7zip does.
A WINRAR file does not reduce a file size but archives it.
There are many types of folder formats that are interesting especially those that consume less space.
